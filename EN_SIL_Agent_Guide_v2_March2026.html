SIL Agent Guide

Version 2.0 · March 2026 · Silence Protocol Initiative
Context
Google's A2A Protocol (v1.0, Linux Foundation) defines the open standard for agent interoperability over HTTP. A2A sends message content as readable JSON. SIL is the content-encryption layer that A2A does not provide. This guide teaches you to deploy SIL agents that communicate in silence — opaque to cloud infrastructure, logging systems, and network monitors.

What is SIL

SIL (Silence Protocol) encodes agent messages into sequences of ancient script symbols — Sumerian cuneiform, Mycenaean Linear B, Cypriot syllabary, Palmyrene script, and others. The encoding is derived from a shared secret via HMAC-SHA256. Without the key, the message is indistinguishable from noise.

Infrastructure neededHTTP server + DNSGit only
Integrity verificationTLS certificateProof-by-nine (public)
Relation—A2A extension (data part)
Compatibility
SIL is expressible as an A2A data part. You can run SIL over A2A transport — agents without SIL see an opaque payload, agents with SIL decode the content. SIL does not break A2A compliance.

Collision guarantee
All 6 alphabets have ≥ 45 symbols. Digits and space are always encoded in CU or LB (≥ 127 symbols). This guarantees zero collisions across all 37 token types at all positions. Verified: 257/257 tokens certain on 257-character message.

A2A Bridge

Use SIL as an A2A content-encryption extension. The SIL payload travels as an A2A data part:

# A2A message with SIL encryption:
{
  "parts": [{
    "data": {
      "x-sil-version":  "1.5",
      "x-sil-encoded":  "𐀀𐀣𐀉𐀊𐎃𐎍...",
      "x-sil-checksum": 9,
      "x-sil-sig":      "a97b187c..."
    },
    "mediaType": "application/x-sil"
  }]
}

# A2A agents without SIL: see opaque data part → ignore
# A2A agents with SIL   : decode content → process

from zmp_v15 import verify9, gkl_ext, symbols_to_sigma_reduced, detect_lang

def verify_integrity(encoded_msg):
    pairs = [encoded_msg[i:i+2] for i in range(0, len(encoded_msg), 2)]
    for pair in pairs:
        lang = detect_lang(pair[0])
        s = symbols_to_sigma_reduced(pair[0], pair[1], lang)
        a, b, ok = integrity_check(s)
        if not ok:
            return False
    return True
